# Square

12s, 1024MB

给定一个二进制数 $s$​ ，但它的若干位置被换成了 `?` 。你需要求出一种把 `?` 替换成 01 的方案，使得替换之后得到一个平方数。

保证方案存在且唯一。保证 $s$ 最高位是 $1$ 。

## 输入格式

第一行一个正整数 $T$ ，表示数据组数。

对于每组数据，一行一个字符串 $s$ 。

## 输出格式

对于每组数据输出替换后的二进制数。

## 样例输入

```
3
1???
1
10??110??00??1000??
```

## 样例输出

```
Case #1: 1001
Case #2: 1
Case #3: 1011110110000100001
```

## 数据范围

$T\le 25, |s|\le 125, \sum[s_i=\texttt{?}]\le 40$
